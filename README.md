# 🗺️ Aquí Nomás — Rutas de Senderismo en Quito

**Aquí Nomás** es una plataforma web estática y completamente responsiva diseñada para entusiastas del senderismo y las actividades al aire libre en Quito, Ecuador. El proyecto centraliza información detallada sobre rutas locales, ofreciendo datos clave como altitud, distancia y niveles de dificultad mediante una interfaz intuitiva, limpia y optimizada para dispositivos móviles.

Desarrollado bajo principios estrictos de **UX/UI (Mobile First)** y gestionado mediante metodologías ágiles.

---

## 🚀 Características Clave

*   **Enfoque Mobile-First:** Interfaz adaptativa optimizada minuciosamente para pantallas móviles y de escritorio.
*   **Fichas Técnicas de Rutas:** Información precisa sobre altitud máxima, desnivel, distancia total y tiempos estimados de caminata en los alrededores de Quito.
*   **UI Fluida y Moderna:** Animaciones sutiles y transiciones de alto rendimiento implementadas con CSS nativo.
*   **Optimización SEO y Performance:** Estructura semántica diseñada para cargar de forma instantánea.
